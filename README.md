# Java_SpringBoot
1. 資料驗證 - Bean Validation、DTO、全局例外處理。
2. 安全性 - 認證/授權、輸入消毒、傳輸與靜態資料加密、稽核日誌。
3. 錯誤處理與日誌 - 集中式日誌、結構化日誌、例外分類與重試策略。
4. 測試與CI/CD - 單元、整合、契約測試；自動化建置與部署管線。
5. 部屬與運維 - 容器化、基礎設施即程式碼、藍綠/滾動部署、回滾機制。
6. 資料遷移與備援 - schema migration、備份/還原、災難復原計畫。
7. 可觀測性 - 指標（Prometheus）、分散式追蹤（Jaeger）、告警與儀表板。
8. 效能與容量規劃 - 負載測試、快取策略、資源配額與自動擴縮。
9. 合規與隱私 - 資料保留、合規審計、GDPR/地方法規考量。
10. 文件與開發者體驗 - - API 文件（OpenAPI）、README、運維 runbook。
11. 入庫 - 資料庫選型、連線池、ORM、資料庫分片與複製。
12. RESTful API - 資源設計、HTTP方法、狀態碼、版本控制、HATEOAS。
13. 聚合 - 聚合根、邊界上下文、事件驅動架構、CQRS。
14. 同步 - REST、gRPC、GraphQL、WebSocket、Server-Sent Events。
15. 非同步 - 消息隊列（RabbitMQ、Kafka）、事件驅動架構、重試與死信隊列。
16. 監控 - 指標（Prometheus）、分散式追蹤（Jaeger）、日誌聚合（ELK Stack）。
